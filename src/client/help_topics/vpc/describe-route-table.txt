NAME

DescribeRouteTables - you can View available route tables.

SYNOPSIS

DescribeRouteTables

DescribeRouteTables [help]


DESCRIPTION


You can view the list of available route tables by running the DescribeRouteTables action.
There are no request parameters available for the DescribeRouteTables action


OPTIONS :

No additional option Required.

EXAMPLES :

jcs vpc DescribeRouteTables
