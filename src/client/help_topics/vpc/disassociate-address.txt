NAME

DisassociateAddress - Disassociate an IP address from an instance.

SYNOPSIS

DisassociateAddress --AssociationId <Association Id>

DisassociateAddress [help]


DESCRIPTION


You can disassociate an IP address from an instance, run the DisassociateAddress action.

Mandatory arguments: --AssociationId


OPTIONS :

--AssociationId  <The unique identifier representing the association between the instance and the RJIL routable IP address>


EXAMPLES :

jcs vpc DisassociateAddress --AssociationId eipassoc-aa7486c3
