NAME

DeleteSubnet - Deletes the specified subnet.

SYNOPSIS

DeleteSubnet --SubnetId <Specified Subnet id>

DeleteSubnet [help]


DESCRIPTION


You can delete subnets whose subnet IP addresses are unallocated using the
DeleteSubnet action. That is, you can delete a subnet if there are no JCS Compute
instances inside it.

Mandatory arguments: --SubnetId 


OPTIONS :

--SubnetId  <The unique identifier of the Subnet to be delete>

EXAMPLES :

jcs vpc DeleteSubnet --SubnetId subnet-9d4a7b6c
